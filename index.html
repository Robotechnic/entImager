Ce projet est une extetion disponible pour les principaux navigateurs de maché qui permet de facilement intégrer des images et des scripts dans les mails de l'ent.
Cette extention en particulier permet d'intégrer des images qui supriment les conversation l'orsqu'une personne lit le message.

Le projet est disponible sur git hub:
https://github.com/Robotechnic/entImager
